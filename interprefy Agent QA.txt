Can I use Interprefy Agent in a customer Teams demo?

Yes absolutely, please remember to request an Interprefy Agent email address for
your demo event (if you don’t have one already) by raising an SD ticket and
supplying the Event ID.

When should I sell the Interprefy Agent solution?
-When the customer needs autonomy, flexibility and/or has a smaller budget that may not match Inject due to remote support costs.

Interprefy Agent is designed for online meetings rather than events, however, if the customer has budget constraints, the Agent could be also used for hybrid sessions instead of Inject (if the customer is ok with the Agent delivery via QR code and URL link).  One Agent email will be needed per room (if several simultaneous sessions).

Currently Interprefy Agent is available only for MS Teams, Zoom, Google meet and Webex.


• Interprefy Agent is normally used by clients that do not want Interprefy support. Though we can provide support if it is required. This should be priced into the quote.
• Interprefy Agentis limited to 10 languages if using auto language switching, and 25 languages if using manual language switching.
• Interprefy Agent can use automatic language switching. It can also use manual language switching. The client will need to be prepared to do the language switching themselves

rather than via Teams/Zoom etc.
